Sobre mim

Sou Gabriel Fernandes, desenvolvedor de sistemas com experiência em JavaScript, TypeScript, HTML, CSS e SQL. Tenho experiência no desenvolvimento de sistemas desktop com Electron e integração com bancos de dados PostgreSQL.

Atualmente, também trabalho com análise de dados utilizando Power BI, criando dashboards dinâmicos e estruturando dados para empresas que precisam transformar informações brutas em insights estratégicos.

Entre meus projetos, destaco o sistema You Time, uma aplicação de agendamentos integrada a banco de dados, e um sistema de gerenciamento de alunos com cadastro, edição e controle de notas.

Meu objetivo é criar soluções eficientes, que unam programação e análise de dados, ajudando empresas e pessoas a otimizarem seu tempo e decisões.
